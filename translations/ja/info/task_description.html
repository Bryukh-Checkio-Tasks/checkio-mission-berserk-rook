古来より存在し、多くの人が少なくとも基本的なルールを理解している チェスをあなたも知っているであろう。. チェスは8行(ランクと呼ばれる1から8の数値からなる)、8列(ファイルと呼ばれるaからhの文字からなる) のチェック柄の盤上で二人で行う戦略ゲームである。 盤上の各マスは文字と数値からなる固有の座標で識別される。 (例, "a1", "h8", "d6")

このミッションでのあなたのルークを見てみよう。 素早い突きと強固な守りを持つ危険な恐怖の駒である。 このルークは縦横に空のマスを無制限に移動可能であるが他の駒を飛び越せない。 他の駒と同様に目標となった敵の駒がいたマスを占めるやり方で捕獲を行う。

このミッションは凶暴化ルーク1つとそれに敵対するルークの軍隊で行う。 凶暴化ルークは敵を捕獲できる場合にのみ移動できる。 手番において目標となる敵の駒がある限りそれを捕獲して移動を継続でき、 捕獲不可となった時点で停止する。

あなたは凶暴化ルークの位置と敵ルークの位置が与えられる。 止まることなく可能な限り多くの敵を捕獲するのが目標である。 凶暴化ルークの各移動は敵の捕獲であること。 敵ルークの最大捕獲数が解答である。

入力: 引数は2つ。凶暴化ルークの位置(string型) と敵ルークの位置(string型のset)

出力: 可能な敵ルークの最大捕獲数(integer型)
